# `jQuery`

## 方法

|            描述            |              示例              |
| :------------------------: | :----------------------------: |
|         获取父元素         |       `$('li').parent()`       |
|       获取所有子元素       |    `$('ul').children('li')`    |
|    在元素里获取后代元素    |      `$('ul').find('li')`      |
|      获取所有兄弟元素      | `$('li:first').siblings('li')` |
| 获取元素之后的所有兄弟元素 |   `$('li:first').nextAll()`    |
| 获取元素之前的所有兄弟元素 |    `$('li:last').prevAll()`    |
|     判断元素是否包含类     |   `$('li').hasClass('cur')`    |
|     获取指定索引的元素     |        `$('li').eq(0)`         |
|                            |                                |
