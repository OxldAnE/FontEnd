### 命令行终端

|     命令     |        功能        |
| :----------: | :----------------: |
|     `cd`     |      移动目录      |
|     `ls`     | 列出所在目录的内容 |
|   `mkdir`    |      创建目录      |
| `rmdir` `-r` |      删除目录      |
|   `touch`    |      创建文件      |
|     `mv`     | 移动文件到指定位置 |
|     `cp`     | 复制文件到指定位置 |
|     `rm`     |   永久性删除文件   |
